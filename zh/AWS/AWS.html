🌥️ AWS S3 雲端同步筆記

本頁記錄 KAO-YU-HSIANG 在本地 VS Code 開發與 AWS S3 同步測試的流程與狀態。

🪜 一、目前架構狀態
  • 本地資料夾：C:\Users\User\Desktop\kao-aws-s3-site
  • S3 Bucket：kao-web-static
  • 區域（Region）：ap-northeast-1（東京）
  • 同步方向：Local → S3 （單向上傳）
🧰 二、同步指令
aws s3 sync "C:\Users\User\Desktop\kao-aws-s3-site" s3://kao-web-static

此指令會將本地端所有更新檔案同步至 S3，若不加 --delete 參數，不會刪除舊檔。

🧠 三、驗證流程
  1. 在 VS Code 中修改或新增檔案（如本頁 KAO_AWS.html）。
  2. 於 PowerShell 執行同步指令。
  3. 前往下方網址檢視是否更新。
🌐 四、S3 網站連結

http://kao-web-static.s3-website-ap-northeast-1.amazonaws.com/zh/KAO/KAO_AWS.html

✅ 五、測試狀態

2025-11-01 測試完成：可由 VS Code 修改本地檔案 → 以 aws s3 sync 上傳 → 網頁即時更新。
